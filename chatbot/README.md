# 🤖 FreshCart AI Support Chatbot

A Python-powered intelligent assistant integrated into the Customer Portal to provide 24/7 automated support.

## ✨ Features

- **Order Status Inquiries**: Real-time integration with the database to track user orders.
- **Policy QA**: Automated answers for shipping, returns, and platform usage.
- **Issue Reporting**: Assists users in filing reports for damaged or missing items.
- **Intelligent Intent Classification**: Uses NLP to understand user queries even with slang or typos.

## 🛠️ Tech Stack

- **Language**: Python 3.9+.
- **Framework**: Flask / FastAPI.
- **NLP**: NLTK / Spacy / Transformers.
- **Integration**: REST API consumed by the React Frontend.

## 🚀 Setup

```bash
cd backend
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
python app.py
```
