# ⚖️ FreshCart Centralized Tax Service

An independent microservice that ensures accurate financial calculations across all FreshCart transactions.

## ✨ Features

- **GST/TCS Calculation**: Real-time calculation of Goods and Services Tax and Tax Collected at Source based on regional rules.
- **Dynamic Fee Management**: Handles platform commissions and gateway fees.
- **Reporting**: Provides consolidated data for the Administrator dashboard.

## 🛠️ Tech Stack

- **Server**: Node.js & Express.
- **Logic**: Mathematical engine optimized for precision (preventing floating point errors).

## 🚀 Setup

```bash
cd backend
npm install
npm run dev
```
